## Using the repo
The way to use the repo is pretty simple. 
1. Copy it your system
   ```bash
   git clone https://github.com/CodeNaveen-in/Java.git
   ```
2. Change to it's directory.
   ```bash
   cd Java
   ```
3. Open it to your IDE (I used VS Code)
   ```bash
   code .
   ```
